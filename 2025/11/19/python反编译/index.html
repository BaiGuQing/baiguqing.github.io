Python反编译

✅ 1. 使用 uncompyle6（最常用）

支持 Python 2.7 ~ 3.7 的 .pyc。

安装

pip install uncompyle6

反编译 .pyc 到 .py

uncompyle6 -o . yourfile.pyc

  • -o . 表示输出到当前目录

  • 会自动生成对应的 .py 文件

✅ 2. 使用 decompyle3（支持 Python 3.7+）

如果你的 Python 版本比较新（例如 Python 3.8, 3.9, 3.10+），应该用这个。

安装

pip install decompyle3

反编译

decompyle3 yourfile.pyc > output.py

✅ 3. 手动识别 pyc 文件版本

如果不确定 pyc 编译时的 Python 版本，可以用：

python -m py_compile --version

或使用 pycdc 自动识别：

✅ 4. 使用 pycdc（另一个反编译器）

pip install pycdc

反编译：

pycdc yourfile.pyc > recovered.py

⚠️ 注意事项（你必须知道）

  1. 只有 你自己写的 或 获得授权的 Python 文件可以反编译
    （否则会涉及法律问题）

  2. 反编译的代码可能不完全准确
    特别是变量名会被优化掉

  3. 反编译必须使用与 .pyc 生成相匹配的 Python 版本
    例如：Python 3.10 生成的 pyc 不能用 Python 3.9 的反编译器解析
